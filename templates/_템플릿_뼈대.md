# 편집 템플릿: (스타일 이름 적기 — 예: 숏폼정보)

> 사용법: 이 파일을 복사해서 스타일별로 이름을 바꿔 쓰세요. (예: 숏폼정보.md, 튜토리얼.md)
> 편집할 때 Claude Code에 "이 영상을 templates/숏폼정보.md 템플릿으로 편집해줘" 라고 말하면 됩니다.
> 1분 샘플로 튜닝이 끝난 뒤, "지금까지 맞춘 규칙을 이 파일에 저장해줘" 라고 하면 AI가 아래 빈칸을 채워줍니다.

## 1. 영상 종류
- 어떤 영상인지: (예: 내 얼굴 나오는 정보 전달 숏폼)
- 화자/말투: (예: 1인, 빠르고 또렷한 톤)

## 2. 출력 스펙
- 길이: (예: 30~60초)
- 비율/해상도: (예: 세로 1080×1920, 30fps)

## 3. 컷 편집 스타일
- 페이스: (예: 빠른 컷, 침묵·군더더기 적극 제거)
- 컷 여백: (예: 타이트하게 / 여유 있게)
- 꼭 살릴 것 / 꼭 뺄 것: (예: 웃음·강조 살리기, 말 더듬 제거)

## 4. 자막 스타일
- 글씨체/크기: (예: 굵은 산세리프, 크게)
- 대소문자/줄나눔: (예: 2단어씩, 강조 위주)
- 위치: (예: 화면 아래 1/3)
- 색/외곽선: (예: 흰 글씨 + 검정 외곽선)

## 5. 색보정
- 톤: (예: 따뜻한 시네마틱 / 자연색 / 없음)

## 5-1. 그림체 (이미지 스타일) ★카테고리마다 다름
- 스타일: (예: 미니멀 플랫 일러스트 / 실사 사진풍 / 3D 렌더 / 지브리풍 / 수채화)
- 색 팔레트: (예: 따뜻한 파스텔 / 비비드 / 모노톤)
- 레퍼런스 이미지: (있으면 경로/링크 — 일관성용, REF-2·3 Elements 방식)
- 이미지 생성 모델: (예: gpt-image-1 / FLUX / Nano Banana)

## 6. 모션그래픽
- 엔진: (예: 간단한 건 PIL, 화려한 건 HyperFrames)
- 색 팔레트: (예: 배경 near-black, 강조 오렌지 #FF5A00)
- 폰트: (예: ___)
- 움직임 느낌: (예: 부드러운 이징, 한 번에 하나씩 등장)

## 7. 레퍼런스
- 참고 영상 링크: (있으면 적기 — 단순·재현 가능·내 형식과 같은 것)

## 8. 주의사항 / 피드백 메모
- (튜닝하면서 반복해서 고친 것들을 여기에 적어두면, 다음에 같은 실수를 안 함)
- 예: 자막 가운데 정렬 꼭 확인 / 그래픽이 자막 가리지 않게

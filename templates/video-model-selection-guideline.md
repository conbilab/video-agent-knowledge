# 영상생성(image-to-video) 모델 선택 지침 — 비용 최적화

> Montage 자동화에서 핵심 컷을 image-to-video로 만들 때, **이미지 종류·중요도에 따라 가성비 모델을 고른다.**
> 제공처: fal.ai (제공 모델 중에서 선택). 가격은 2026 기준 대략치 — 변동 가능, fal 가격표로 재확인.
> 대원칙: **핵심 컷 2~3개만 영상생성, 나머지는 Remotion**(지침 A-8). 그 핵심 컷 안에서도 아래로 모델을 골라 비용 최적화.

## 1. 모델별 강점 + 비용 (대략)
| 모델 | 강점 | 비용(대략) | 한줄 |
|---|---|---|---|
| **Kling 3.0** | 실사·인물 일관성, 1080p | **~$0.11~0.17/초** | ⭐ **기본값(가성비 최고)** |
| **Hailuo 2.3** | 인물(표정·몸동작·미세표정) 최강, 빠름 | ~$0.49/클립 | 사람 중심 컷 |
| **PixVerse v5.5** | 2D·일러스트·애니·스타일·VFX 전환 | ~$0.15~0.4/클립 | 스타일라이즈드 |
| **Seedance 2.0** | 올라운드 최고·멀티이미지(9장)/오디오 참조·통합오디오 | ~$0.24~0.30/초(720p) | 비쌈 — 중요컷·멀티참조만 |
| **Veo 3.1** | 최고 시네마틱 실사 + 네이티브 오디오 + 4K | 매우 비쌈(Kling의 ~10x) | 결정적 히어로 1컷만 |

## 2. 이미지 종류별 모델 라우팅 (기본 규칙)
- **실사 일반 컷** → **Kling 3.0** (기본)
- **실사 + 인물/표정/몸동작 중심** → **Hailuo 2.3** (싸고 인물 최강) 또는 Kling
- **2D·일러스트·애니메이션·스타일라이즈드·화려한 전환** → **PixVerse v5.5**
- **3D 렌더 느낌** → **Kling 3.0** 또는 Seedance
- **올라운드 최고 품질 / 여러 참조 이미지 결합 / 통합 오디오 필요** → **Seedance 2.0** (비쌈, 꼭 필요할 때만)
- **단 하나의 영화급 히어로 컷(오디오 포함, 4K)** → **Veo 3.1** (제일 비쌈, 영상당 1컷만)

## 3. 비용 최적화 규칙 (자동 적용 — 매번 지시 불요)
1. **기본은 Kling 3.0.** 대부분 핵심 컷은 이걸로.
2. **사람 표정·몸동작이 주인공** → Hailuo 2.3.
3. **2D/애니/스타일** → PixVerse.
4. **Seedance 2.0은 "전체가 다 최고급이어야 할 때"만** (비쌈). 애매하면 Kling.
5. **Veo 3.1은 진짜 결정적 1컷만** (제일 비쌈).
6. **클립은 짧게**(초당 과금 → 짧을수록 쌈), **720p로 충분하면 720p**(불필요하게 1080p/4K 금지).
7. 핵심 컷만 영상생성, 나머지 Remotion.
8. 같은 컷을 여러 모델로 중복 생성하지 말 것(테스트는 1개만).

## 4. 판단 흐름 (에이전트용)
```
이 컷, image-to-video가 꼭 필요한가? (아니면 Remotion)
 └ 필요 → 이미지 종류는?
     ├ 2D/애니/스타일 → PixVerse
     ├ 실사 + 인물 주연 → Hailuo
     ├ 실사 일반/3D → Kling 3.0 (기본)
     ├ 멀티참조·통합오디오·올라운드 최고 → Seedance 2.0 (비쌈, 정당할 때만)
     └ 영화급 결정적 1컷 → Veo 3.1 (가장 비쌈, 1컷만)
 → 길이 짧게, 720p 기본, 1컷 1회만 생성
```

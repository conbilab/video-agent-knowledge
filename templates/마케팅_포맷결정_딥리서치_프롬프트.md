# 딥리서치 프롬프트 — 숏폼 마케팅 "비주얼 포맷 결정" 범용 프레임워크

> 용도: ChatGPT Deep Research에 붙여넣어, **어떤 업종·주제든 적용 가능한** "장면별 비주얼 포맷 결정" 프레임워크를 근거와 함께 받는다.
> 구조: 이건 **범용(universal)** 프레임워크. 개별 사업(세차고수 등)은 이걸 상속해 **별도 템플릿**으로 얹는다.

---

당신은 **숏폼 영상 마케팅 전략가 겸 비주얼 콘텐츠 리서처**입니다. 아래에 대해 **딥리서치**를 수행하고, **업종에 상관없이 적용 가능한 "장면별 비주얼 포맷 결정 프레임워크"** 를 근거와 함께 만들어 주세요. (특정 업종에 한정하지 말고 범용으로. 단, 업종별로 어떻게 달라지는지의 "조정 규칙"은 포함.)

## 배경/맥락
- 대상: **AI로 숏폼 마케팅·정보 영상을 대량 제작하는 1인/소규모 사업자**(유튜브 쇼츠, 인스타 릴스, 틱톡). AI 생성 이미지 + image-to-video + AI 나레이션 파이프라인.
- 여러 업종에 두루 쓸 **범용 의사결정 기준**이 필요. (교육·정보, 제품·이커머스, 지역 오프라인 서비스, 식음료, 뷰티·패션, 부동산, B2B 등 다양)
- 장면마다 고를 수 있는 **제작 옵션 4가지**:
  1. **실사 AI 이미지**(정지, 사실적 사진풍)
  2. **일러스트/2D**(인포그래픽·플랫 일러스트)
  3. **AI 영상생성**(image-to-video, 진짜 모션 — 가장 비쌈)
  4. **정지 이미지 + 카메라 모션**(켄번즈·줌·패럴랙스 — 가장 저렴)
  - 비용: 옵션3 ≫ 옵션1≈2 > 옵션4
- 목표: **마케팅 효과 최대, 비용 최소.** 핵심 컷만 비싼 옵션, 나머지는 저렴하게.

## 리서치해서 답할 질문 (범용 기준)
1. **포맷별 효과 원리**: 실사 vs 일러스트, "진짜 모션 영상" vs "정지+켄번즈" — 각각 언제 더 효과적인가? (주목·이해·신뢰·시청유지·전환 관점, 근거/연구/사례)
2. **마케팅 목표별**: 인지도(reach) / 전문성·신뢰 / 전환(클릭·방문·구매·문의) — 목표마다 유리한 포맷.
3. **콘텐츠 유형별**: 증명·시연(비포애프터·결과·데모) / 개념 설명·교육 / 감성·스토리·브랜딩 / 제품 소개 / 데이터·수치 — 각각 최적 포맷.
4. **장면 역할별**: 후킹(첫 1~3초) / 본론 설명 / 증거 제시 / 전환(CTA) — 역할마다 어떤 포맷이 강한가.
5. **업종 카테고리별 조정 규칙**(일반화): 지역 서비스 / 이커머스·제품 / 교육·정보 / 식음료 / 뷰티·패션 / 부동산 / B2B 등 — 업종 성격에 따라 포맷 선택이 어떻게 달라지나. (예: "결과를 눈으로 증명해야 하는 업종은 실사 비중↑")
6. **AI 이미지 신뢰 인식**: 마케팅에서 "AI 티 나는 이미지"가 신뢰를 해치는 경우/업종은? 언제 실제 촬영본을 써야 하나? AI 생성 표시(공시) 영향.
7. **플랫폼 차이**: 유튜브 쇼츠 / 인스타 릴스 / 틱톡 — 포맷·길이·후킹·자막 관습.
8. **비용 대비 효과(ROI)**: 어떤 포맷이 노력·비용 대비 임팩트가 큰가. "비싼 모션 영상"을 꼭 써야 하는 경우 vs 정지+모션으로 충분한 경우.

## 출력 형식 (그대로 지침으로 쓸 수 있게)
1. **핵심 요약**(한 페이지) — 포맷 선택의 대원칙 5~7개.
2. **범용 결정 매트릭스(표)**: [콘텐츠 유형 / 마케팅 목표] × [추천 포맷 1·2순위 + 한 줄 이유].
3. **장면 역할별 규칙**: 후킹컷 / 설명컷 / 증명·시연컷 / 전환(CTA)컷 → 추천 포맷.
4. **업종 카테고리별 조정 가이드(표)**: 업종 유형 → 포맷 비중·강조점이 어떻게 달라지는지(일반화). 개별 사업이 이걸 보고 자기 템플릿을 만들 수 있게.
5. **비용-효과 의사결정 흐름**: "이 장면에 비싼 영상생성을 써야 하나?" 판단 플로우.
6. **안티패턴(하지 말 것)**: 흔한 실수·역효과.
7. 각 권고에 **근거·출처(URL)** 와 **확실성 표시**(높음/중간/낮음).

## 제약
- **특정 업종에 한정하지 말 것.** 범용 원칙 + 업종별 조정 규칙으로.
- 한국 시장·한국어 시청자를 우선 고려하되, 글로벌 베스트프랙티스도 참고.
- "고품질 영상이 좋다" 식 막연한 결론 금지 — **구체적·실행 가능**하게. **표·체크리스트 위주.**
- 추측은 추측이라 명시하고 출처 있는 사실과 구분. 효과 수치는 출처 있을 때만 인용.

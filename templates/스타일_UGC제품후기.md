# 스타일 템플릿: UGC (제품 후기형)

> 사용법: 제품마다 폴더를 만들어 **사진 + 정보**를 넣고,
> "이 폴더(ugc/제품명)로 UGC 영상 만들어줘" 하면 읽어서 제작.
> 보편 규칙(CLAUDE.md: 숫자 한글, 자막 끊기, 음성, 검수)은 자동 상속.

## ★ 입력 폴더 구조 (재빈님이 채움)
```
ugc/<제품명>/
├── assets/        ← 실제 제품 사진 여러 장 (다양한 각도/사용 장면)
└── info.md        ← 제품 정보 (아래 양식)
```

### info.md 양식 (이대로 채우면 됨)
```
# 제품명:
# 한줄 소개:
# 핵심 효능·특징 (3가지):
- 1)
- 2)
- 3)
# 사용법 (단계):
- 1)
- 2)
# 타겟 (누구에게):
# 후킹 멘트 / 강조 포인트:
# 톤: (예: 친근한 후기 / 발견의 놀라움 / 솔직 리뷰)
```

## 제작 방식 (에이전트가 자동으로)
1. `info.md` + `assets/` 사진을 **읽는다**
2. **구어체 후기 대본**을 쓴다 (1인칭, "이거 진짜 써봤는데…" 같은 자연스러운 추천 톤)
3. **재빈님이 준 실제 제품 사진을 영상에 그대로 사용** (AI로 제품을 새로 만들지 말 것 — 진짜 사진을 이미지→모션으로)
4. 필요 시 "사용 장면" 정도만 보조로 생성
5. 나레이션 + 자막 + 음악 입혀 완성

## 출력 스펙
- 9:16 세로, 20~40초

## 그림체 / 룩
- **UGC 특유의 리얼·캐주얼 룩**: 과한 광고티 X, 자연광, 핸드헬드 느낌
- 제품은 **실제 사진 유지**(왜곡 금지)
- 자막: 캐주얼·통통 튀는 느낌, 강조색
- 전체 톤: "친구가 진심으로 추천하는" 느낌

## 모션그래픽
- 가벼운 줌/핸드헬드 흔들림, 통통 튀는 자막, 빠른 컷
- 엔진: Remotion

## 음악
- 트렌디·경쾌한 로열티프리 (틱톡/릴스 무드)

## 음성

## 주의 / 체크
- **제품 사진 왜곡·재생성 금지** (실제 제품 그대로)
- 숫자 한글(오 분), 자막=대본 일치, IG 세이프존
- 과장 광고 표현 주의(효능 단정 X)

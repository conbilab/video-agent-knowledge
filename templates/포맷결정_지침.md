# 비주얼 포맷 결정 지침 (범용) — 장면마다 무엇으로 만들까

> 모든 사업·주제 공통. 에이전트가 **대본 단계에서 장면마다 포맷을 이유와 함께 제안**할 때 이 지침을 따른다.
> 개별 사업(세차고수 등)은 이 위에 자기 템플릿을 얹는다.
> 출처: GPT 딥리서치(2026). 신뢰도 표기: 높음/중간.

## 0. 포맷 4종 + 비용
1. **실사 AI 이미지**(정지) — 신뢰·감정몰입
2. **일러스트/2D** — 복잡·추상 개념 설명
3. **AI 영상생성(image-to-video)** — 역동·극적·시연 (가장 비쌈)
4. **정지 이미지 + 카메라 모션(켄번즈/줌)** — 저비용 유지
- 비용: 3 ≫ 1≈2 > 4

## 1. 포맷 선택 대원칙
- **실사** = 신뢰·진정성·감정연결 (사람·브랜드·증언·결과). **일러스트/2D** = 복잡/추상 개념을 명료하게. **영상(#3)** = 변화·시연·극적 순간. **정지+켄번즈(#4)** = 비용 절감하며 시선 유지.
- ★ **사람·증언·서비스 제공자는 "실제 촬영"이 원칙. AI는 배경·개념·연출 보조로.** (소비자 90%가 AI 여부 알고 싶어 하고, 98%가 진정성을 신뢰의 필수로 봄 — 높음)
- **첫 3초 후킹** 필수 (틱톡 시청자 71%가 첫 3초에 지속 여부 결정 — 중간). 느린 인트로 금지.
- **9:16 세로, 15~30초(교육형 최대 60초)**, 자막·BGM·빠른 컷. (중간)

## 2. 결정 매트릭스 (목표·유형 → 포맷)
| 콘텐츠 유형 / 목표 | 1순위 | 2순위 |
|---|---|---|
| 증명·시연(Proof/Demo) | **영상(#3)** 전후 과정 동적 | 정지+켄번즈(#4) 전/후 사진 |
| 개념설명·교육 | **일러스트/2D(#2)** 개념 명료 (높음) | 영상(#3) 동적 그래픽 |
| 감성·브랜딩 | **실사(#1)** 진정성·감정 (높음) | 영상(#3) 필름 스타일 |
| 제품/장비 소개 | **영상(#3)** 성능 입체 시연 | 실사(#1) 실물·결과 사진 |
| 인지도(Awareness) | **영상(#3)** 역동·도달↑ | 정지+켄번즈(#4) 저비용 A/B |
| 전문성·신뢰 | **실사(#1)** 실제 전문가·고객 (높음) | 영상(#3) 과학적 설명·실험 |
| 전환(Conversion) | **영상(#3)** CTA 포함 | 실사(#1) 로고·연락처 컷 |

## 3. 장면 역할별 규칙
- **후킹 컷**: 첫 3초 강렬한 비주얼(변화·질문·큰 움직임). → 영상(#3) 또는 정지+켄번즈(#4) 극적 패닝.
- **증명·시연 컷**: 전/후·과정·체험을 리얼하게. → 영상(#3) 또는 실사(#1). 자료화면틱하지 말고 리얼리티.
- **설명·교육 컷**: 원리·기능. → 일러스트/2D(#2) (높음) + 핵심 키워드 자막.
- **마무리·CTA 컷**: 로고·메시지·연락처. → 고화질 실사(#1) + 행동유도 자막. 과한 모션보다 명확성.

## 4. AI vs 실제 촬영 — 신뢰 규칙 ★
- **실제 촬영을 써야 하는 것**: 사람(직원·고객), 고객 후기·증언, 서비스 제공 장면, "진짜 결과"(비포애프터 실물).
- **AI 생성 OK**: 배경, 개념·원리 그래픽, 연출컷, 정보 비주얼.
- AI로 사람을 무분별하게 합성하면 **신뢰 급락**. AI 사용 시 공시 라벨.

## 5. 비용-효과 결정 흐름
```
이 장면이 "설득의 핵심"(증명·드라마틱 전후·제품 성능)인가?
 ├ 예 → 예산 투입: 영상(#3) 또는 실제 촬영/실사(#1)
 └ 아니오 → 저비용: 정지+켄번즈(#4) 또는 일러스트(#2)
설명·개념 → 일러스트(#2) / 사람·신뢰 → 실제 촬영
```
핵심 컷만 비싼 포맷, 나머지는 #4/#2로. (ROI 극대화)

## 6. 안티패턴 (하지 말 것)
- 첫 3초 후킹 없는 **느린 인트로**.
- **AI 인물 남용** — 특히 후기·직원 컷에 AI 티 나는 사람 → 신뢰 역효과.
- **저품질·정지 반복** — 같은 정지 이미지 단순 반복, 심심한 컷 나열.
- **텍스트 과다** — 핵심 단어·숫자만, 나머지는 나레이션/자막 보조.
- **플랫폼 규격 무시** — 9:16 아님, 트렌드 음악/포맷 무시.

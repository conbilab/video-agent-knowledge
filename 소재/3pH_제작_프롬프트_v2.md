[확정 대본 — 7장면]  (나레이션 / 자막 / 포맷)
1. "비싼 코팅 했는데, 차가 왜 금방 더러워질까요?" / 비싼 코팅 했는데 왜 금방 더러워질까? / **실사 + image-to-video(핵심컷)** — 얼룩진 도장면 위 물방울이 흐름
2. "문제는 코팅이 아니라 세차일 수 있어요." / 문제는 코팅이 아니라 세차 / 실사(정지+켄번즈)
3. "자동차 오염은 두 가지예요. 먼지 같은 이물질, 빗물 자국 같은 얼룩." / 오염은 2가지 · 이물질 / 얼룩 / **일러스트·2D(개념)**
4. "이물질은 알칼리성, 얼룩은 산성, 마무리는 중성. 산도가 다른 세 약품을 순서대로 쓰는 게 쓰리 피에이치 세차예요." / 알칼리 → 산성 → 중성 = 3pH / **일러스트·2D(개념)**
5. "물로 헹궈도 안 지워지던 미네랄 얼룩까지 녹여서, 도장면이 진짜 깨끗해져요." / 미네랄 얼룩까지 싹 / **실사 + image-to-video(핵심컷)** — 더러운 면→깨끗하게, 물이 매끄럽게 흐름
6. "흠집은 줄고, 코팅은 더 오래가고요." / 흠집 ↓ 코팅 수명 ↑ / 실사(정지+켄번즈)
7. "기억하세요. 코팅보다 세정이 먼저입니다." / 코팅보다 세정이 먼저 / 실사 + 세차고수 가벼운 아웃트로

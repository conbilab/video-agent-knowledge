# 3pH 세차 — 핵심 정리 (영상 소재용)

> 출처: 세알남 유튜브 "전세계 세차 방법이 3pH로 변하고 있습니다" (셀프세차 교육 영상)
> 용도: 3pH의 원리·장점을 우리(세차고수) 콘텐츠 소재로 정리. ※ 브랜드(마프라·라보코스메티카 등)는 세알남 것 → 우리 영상은 "개념·장점" 중심으로.

## 1. 한 줄 정의
**3pH 세차 = 산도(pH)가 다른 약품 3가지를 순서대로 써서, 차에 흠집 없이 오염을 화학적으로 제거하는 세차법.**

## 2. 자동차 오염은 2종류
- **이물질** — 먼지·흑먼지 (손/물로 떨어지는 것) → **알칼리성**으로 제거
- **얼룩** — 미네랄·빗물자국 (문질러야/녹여야 하는 것) → **산성**으로 제거

## 3. 3단계 (순서가 중요)
1. **고압수 프리워시** — 큰 이물질·모래알 먼저 떨어뜨림 (손대기 전, 미세 스크래치 방지)
2. **① 알칼리성 프리워시** — 먼지·이물질을 불려서 제거 → 헹굼
3. **② 산성 프리워시** — 미네랄·빗물 얼룩을 녹여 제거 → (중성 샴푸로 미트질)
4. **③ 중성 카샴푸** — 마무리 세정 → 헹굼 → 드라잉
> ⚠️ 알칼리성과 산성을 섞으면 안 됨. 순서대로.

## 4. 왜 3pH가 좋은가 (핵심 장점) ★영상 메인
1. **미세 스크래치 방지** — 오염물을 화학으로 먼저 떼어내니, 문질러도 모래알 흠집이 적다
2. **물자국·미네랄 얼룩까지 완전 제거** — 알칼리만으론 못 지우는 얼룩을 산성이 녹임 → 도장면에 물 흐름을 방해하는 게 없어짐(= 진짜 깨끗)
3. **코팅이 제대로 입혀짐** — 깨끗한 면 위에 코팅해야 본딩·광택·지속력이 산다 ("코팅보다 세정이 더 중요")
4. **코팅층 데미지 최소화** — 코팅 위 이물질·미네랄만 골라 제거 → 유리막/세라믹 코팅 위에도 안전, 코팅 더 오래감
5. **장기 관리가 쉬워짐** — 묵은때 축적이 줄어 폴리싱 주기가 길어짐(예: 4년 폴리싱 없이 깔끔 유지)
6. **여러 제품을 대체** — 벌레제거제·워터스팟리무버·휠클리너 따로 필요 없음

## 5. 상황별 (몇 pH가 필요한가)
- **많이 더러운 차** → 3pH 풀세차(알칼리→산성→중성)
- **비만 맞은 차(얼룩 위주)** → 산성만 = 2pH
- **먼지만 앉은 차** → 알칼리/물만 = 1pH
- **코팅 잘 된 새 차** → 물 + 중성 샴푸로 가볍게
> 핵심: "세차 원리를 알면 내 차에 뭐가 필요한지 스스로 판단할 수 있다."

## 6. 한 줄 메시지(영상 마무리용)
**"코팅보다 세정이 먼저다. 잘 씻어야 코팅도, 차 상태도 오래 간다."**

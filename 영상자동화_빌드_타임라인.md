# 영상 자동화 에이전트 — 빌드 타임라인 (build in public)

> **목표:** 내 사업들(세차고수 등) 숏폼 마케팅·교육 영상을 **AI로 자동 제작**하는 시스템.
> 한 줄로: *폰에서 "이런 영상" → 기획안 → 승인 → 완성 영상.*
> **나는 개발 초보.** 아래는 그 과정에서 막힌 것 · 실패한 것 · 성공한 것 전부.

---

## 0. 시작 — 만들기 전에 모으기
- "처음부터 만들지 말고 **좋은 오픈소스 위에** 쌓자." → 정보부터 수집.
- 유튜브·웹의 영상 자동화 레퍼런스를 계속 모아 **지식베이스** 구축.
- ⛔ **윤리 선택:** AI로 실존 인물 딥페이크 만드는 류(예: AI OnlyFans)는 거절. 동의·신뢰가 먼저.

## 1. 첫 벽 — 코드가 아니라 '환경'이었다
- ❌ 명령어를 엉뚱한 창(파워셸 vs 리눅스)에 붙여넣음 → 프롬프트 색(초록/흰) 구분부터 배움.
- ❌ `uv` 가상환경에 pip이 없음 → `uv pip install`로 전환.
- ❌ 바탕화면에 저장한 파일이 사라짐(원드라이브가 데스크톱을 옮겨감) → 작업 폴더를 D드라이브로 고정.
- ✅ **배움:** "되는 환경"을 먼저 만드는 게 8할.

## 2. 두뇌 설계 — 지침이 많으면 AI가 못 알아듣는다
- ✅ **2층 구조:** 항상 읽는 짧은 규칙 + **이거 할 때만 읽는** 상세 지침 + 사업별 스타일 템플릿.
- ❌ 처음에 "세차 전용"으로 만들었다가 → *"범용성 먼저, 디테일은 그다음!"* 교정. 범용 프레임워크 → 사업별 템플릿 순서로.
- ❌ "지침을 파일로 저장 = 설치"인 줄 앎. 아니었음. 에이전트에 실제로 넣는 단계가 따로 있었다.
- ❌ 긴 지침을 요약하다 핵심 템플릿을 빠뜨림 → 들킴 😅 → 완전판으로 복구.

## 3. 한국어 목소리 — 자연스러움과의 싸움
- ❌ 기본 보이스+모델 조합 = 어색함. 최신 모델은 더 불안정.
- ❌ 무료 플랜이 API를 막음(402) → 유료 전환.
- ✅ **특정 보이스 + 다국어 모델 + 음성 세팅값 튜닝** = "훨씬 좋은데." 합격.

## 4. 이미지 — 한글이 깨지면 끝
- ❌ 이미지 모델이 한글을 `Protectivective` 처럼 깨뜨림 → *"한글 깨지는 건 절대 안 돼."*
- ✅ 결정: **한글 잘 쓰는 상위 모델**로 교체(비용 더 들어도 OK). 핵심 자막은 아예 **별도 레이어**로 얹기.
- ✅ **실사 프롬프트 가이드라인** 정립 — 실제 카메라·조명·질감·결함까지 적어야 'AI 티'가 안 난다.

## 5. AI 영상 생성 — 돈이 '선불'이었다
- ❌ 영상 생성 API가 403 → 알고 보니 **잔액 $0** (선불인 줄 몰랐음).
- ✅ 충전 + 자동충전 설정 (자동충전도 "한도 < 충전액" 규칙에 한 번 막혔다 해결).
- ✅ **비용 전략:** 핵심 컷 2~3개만 진짜 영상생성(비쌈), 나머지는 정지+카메라 모션. ROI 극대화.

## 6. 무엇을 무엇으로 만들까 — 포맷 결정
- ✅ GPT 딥리서치로 마케팅 근거 수집 → **"장면별 포맷 결정" 범용 지침**화.
- ✅ 원칙: 사람·후기·결과 = **실제 촬영**, 배경·개념·원리 = AI. (소비자는 진정성을 신뢰)
- ✅ 첫 영상(3pH 세차 원리): **대본 먼저 사람이 확인 → 제작.** 첫 버전은 (영상생성 막혀서) 전부 정지 기반 → 지금 개선판(실사 + 핵심컷 영상) 준비 중.

## 7. 지금 — 홈서버로 이사 중
- *"어차피 퀄리티는 계속 업그레이드해야 하니, 홈서버에 올려 **언제 어디서든** 테스트하자."*
- ✅ "open claw"의 정체 = **OpenClaw** (폰 챗앱 ↔ AI 에이전트 다리). 텔레그램으로 "이런 영상" 보내면 기획안 받고 승인→제작. **내가 원하던 게 이미 도구로 존재.**
- 구성: 폰 →(Tailscale)→ 홈서버 [ **OpenClaw**(다리) → **Claude Code**(두뇌) → **OpenMontage**(제작) ] → 결과 폰으로.
- 서버 점검: Python·Node·ffmpeg·Docker·Claude Code 다 있고 **`uv` 하나만** 빠져서 설치. 거의 준비 완료.
